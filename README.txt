
# PRUEBA

Ejercicio 2: PetStore



## Requisitos

Para ejecutar el proyecto es necesario tener instalado lo siguiente:


  - JDK 11
  - Intellij-Idea Comunity o Ultimate



## Ejecucion
- Para ejecutar el proyecto abrirlo con IntellijIdea
- Ubicarse en la clase KarateRunner.java
- Ejecutar el test General
- Esto ejecutar√° todos los tests en el orden especificado.
- Tambien se puede ejecutar cada escenario de manera independiente, para ello debe ejecutar unicamente el KarateRunner.testExerciseX, donde 'X' corresponde al numero de la tag determinada.


## Reporte

Al ejecutar cada uno de los escenarios puede acceder al reporte grafico desde el log en la terminal o en la ruta <b>target/karate-reports/karate-summary.html<b>
